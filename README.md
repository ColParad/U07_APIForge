# APIForge

**A privacy-first, offline-first desktop client for working with HTTP APIs.**

This repository is the public home of **APIForge Core**. It is being initialized as a clean public repository so that only reviewed, explicitly approved open-source materials are published here.

## Current status

The public source export is not yet published. The project is currently separating the open-source core distribution from optional separately distributed product capabilities and completing a publication review.

No internal architecture, deployment, licensing-service implementation, security audit, or release-control materials belong in this repository.

## Intended public scope

The public Core distribution is intended to cover developer-facing local API client functionality, including:

- building and sending HTTP requests;
- organising local collections and environments;
- viewing responses;
- importing and exporting supported request formats;
- desktop operation without requiring a cloud account for core workflows.

The exact source tree will be published only after it is independently buildable and its dependency and licensing boundary has been validated.

## Repository policy

Public additions to this repository are allowlisted: files are published only after review for licensing, secrets, internal-only documentation, and commercial-service coupling.

See [`docs/public/OPEN_CORE_SCOPE.md`](docs/public/OPEN_CORE_SCOPE.md) for the publication boundary.
